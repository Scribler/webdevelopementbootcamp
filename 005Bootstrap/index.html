This implementation of bootstrap is using a cdn and the bootstrap starter template that integrates the javascript needed in order to make all the functionality in bootstrap work. This might be something worth putting in a snippet later if it ends up being something that I use a lot.

*WARNING*

Use the CDN version NOT the downloadable version.

Why?

Because most people will have bootstrap cashed, and so this will save the reader from having to redownload it for your website.

If you use your own version of bootstrap, the user's computer will not know if it is already cashed, and so your website will always be slower.
